## 💻 安装步骤

### 自动安装（推荐所有用户）

**Windows 用户：**

1. **双击** `RUN_DASHBOARD.bat`
2. **等待**自动检查和安装：
   - ✅ 检查Python是否安装

   - 勾选：只显示有交通数据的房源
   - 确保结果准确

**使用示例：**

**场景1：在Basel工作，想在法国租房**

## 🔧 常见问题

### 问题1：仪表板打不开

**解决方案：**
1. **双击** `RUN_DASHBOARD.bat`
2. **等待**自动修复依赖
3. **如果**提示Python未安装：
   - 访问 python.org
   - 下载Python 3.8+
   - 安装时勾选 "Add to PATH"

---

### 问题2：看不到房源

**解决方案：**
1. **检查**筛选条件是否太严格
2. **尝试** "不限" 可入住时间
3. **扩大**价格和面积范围
4. **点击** "扫描Bien'ici" 获取新房源

---

### 问题3：Python安装失败

**解决方案：**
```bash
# 1. 确保下载了正确的版本
# Windows: 64位或32位

# 2. 安装时必须勾选
✅ Add Python to PATH

# 3. 安装后验证
python --version

# 4. 如果还有问题
pip install --upgrade pip
```

---

### 问题4：浏览器不打开

**解决方案：**
1. **手动访问** http://127.0.0.1:8501
2. **尝试**其他浏览器（Chrome、Firefox、Edge）
3. **检查**防火墙设置
4. **确保**端口8501未被占用

---

### 问题5：依赖安装失败

**解决方案：**
```bash
# 1. 升级pip
python -m pip install --upgrade pip

# 2. 清理缓存
pip cache purge

# 3. 重新安装
pip install -r requirements.txt

# 4. 如果还是失败
pip install streamlit pandas sqlalchemy requests beautifulsoup4
```

## 📞 获取帮助

### 常见问题

**Q: 应该多久扫描一次新房源？**
A: **每周一次**最佳。新房源每天出现，好房源很快被租。

**Q: 应该用"3个月内"还是"4个月内"？**
A: **3个月**适合有灵活性的人，**4个月**提供更多选择。

**Q: 为什么有些房源没有显示可入住日期？**
A: 不是所有房源都提到可入住时间。使用"不限"可以看到这些。

**Q: 如何保存我的搜索？**
A: 使用 **⭐ 收藏**功能保存感兴趣的房源。

**Q: 价格有多准确？**
A: 价格来自原始房源。**始终在原始网站验证**。
